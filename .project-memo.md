# 项目笔记

## 踩坑记录

**Glance Widget API 限制：**
- `Box` 必须有 `content = {}` 即使为空
- `fillMaxWidth(fraction)` 不存在，用 `width(dp)` 或重叠 Box 实现进度条
- Column 内不能用 `Modifier.align()`，改用 `fillMaxWidth()`

**WebView 登录：**
- 京东重定向后 cookie 可能在父域（`.jdcloud.com`），多域名尝试提取
- `setAcceptThirdPartyCookies` 在部分 SDK 版本不暴露，需去掉
- 自动回调不靠谱，改用用户手动点"提取 Cookie"按钮

**GitHub Actions：**
- Node.js 20 → 24 升级截止 2026-06-02，设置 `FORCE_JAVASCRIPT_ACTIONS_TO_NODE24=true`
- Workflow 没有 `workflow_dispatch` trigger 需手动 run 时要在 on 列表加上
- 首次构建 Gradle 下载依赖约 3-4 分钟
